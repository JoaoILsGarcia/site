Projetos

Portifólio

Este site representa mais do que algumas linhas de código: ele marca o meu primeiro passo no desenvolvimento web. Criado exclusivamente com HTML, sem frameworks ou bibliotecas externas, cada elemento foi pensado e estruturado do zero, reforçando minha base técnica e compreensão real da web. A escolha pelo HTML puro demonstra domínio dos fundamentos essenciais, como semântica, organização de conteúdo e estrutura correta das páginas — pilares indispensáveis para qualquer desenvolvedor sólido. Aqui, não há atalhos: há aprendizado, lógica e intenção em cada tag utilizada. Mesmo sendo um projeto inicial, este site reflete dedicação, curiosidade e vontade de evoluir. Ele prova que grandes projetos começam simples, mas com propósito. É o início de uma jornada focada em crescimento constante, boas práticas e paixão por tecnologia. Este não é apenas um site. É o começo da minha história como desenvolvedor.
